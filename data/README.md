# Dataset

https://www.kaggle.com/datasets/mlg-ulb/creditcardfraud 

This project uses the Credit Card Fraud Detection dataset from Kaggle. The dataset is not included in this repository due to licensing and file size considerations.

Download creditcard.csv from the dataset source and place it inside this folder:

data/
└── creditcard.csv

The notebook expects the dataset to be located at:

data/creditcard.csv

Once the dataset has been added, the notebook can be run without any further changes
